We solve the system
[
\begin{cases}
2x + 3y = 12\
x - y = 1
\end{cases}
]

From the second equation:
[
x - y = 1 ;\Rightarrow; x = 1 + y
]

Substitute into the first equation:
[
2(1+y) + 3y = 12
]
[
2 + 2y + 3y = 12
]
[
2 + 5y = 12
]
[
5y = 10
]
[
y = 2
]

Now compute (x):
[
x = 1 + y = 1 + 2 = 3
]

**Answer:** (\boxed{x=3,; y=2}).
